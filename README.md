# votacao-de-personagens
O site apresenta um Sistema de Votação de Personagens criado para permitir que a comunidade escolha quais heróis, inventores ou guardiões terão destaque em eventos, capítulos ou missões de uma narrativa. Com um visual envolvente e imagens de alta qualidade, cada personagem é apresentado com sua história, personalidade e habilidades, permitindo que o usuário conheça profundamente suas motivações e características antes de votar.
